# 🤖 Bybit Grid Bot V2 — Long Only

Автоматизированный лонг-Only grid trading бот для Bybit Testnet с AI-сигналами (LSTM) и управлением через Telegram.

## 📋 Возможности

### 🔄 Grid Trading
- **Только лонг** — покупка ниже цены, продажа выше. Нет шортов → нет риска бесконечного убытка при росте
- **Динамический диапазон** — сетка строится по ATR(14) на 4h свечах, автоматически подстраивается под волатильность
- **Trailing Up/Down** — сетка следует за ценой при выходе за границы диапазона
- **Автоматическое восстановление** — при исполнении BUY ордера автоматически ставится SELL на следующем уровне

### 🧠 AI / LSTM
- **Ансамбль из 3 моделей** — GRU-сети с batch normalization и dropout
- **15 технических фичей** — RSI, StochRSI, Williams %R, CCI, MACD, Bollinger Bands, OBV, ATR, Momentum и др.
- **Реальные данные объёма** — из 15-минутных свечей Bybit
- **Динамическая корректировка** — при медвежьем сигнале qty уменьшается на 50%, при STRONG_SELL BUY ордера не ставятся

### 🛡 Защиты
- **Глобальный Stop-Loss** — закрытие всех позиций при unrealized убытке > порога
- **Глобальный Take-Profit** — фиксация прибыли при realized PnL > порога
- **Минимальный баланс** — остановка бота при падении баланса ниже минимума
- **Rate limit** — задержки между ордерами, retry при ошибках API

### 📱 Telegram
- **Управление ботом** — старт, стоп, экстренная остановка
- **Мониторинг** — статус, баланс, параметры, AI сигнал
- **Уведомления** — исполнение ордеров, сдвиг сетки, срабатывание защит
- **Настройка** — изменение параметров на лету через `/set`

## 🚀 Установка

### Требования
- Python 3.10+
- Аккаунт Bybit (тестовый или реальный)
- Telegram Bot Token

### 1. Клонирование
```bash
git clone https://github.com/yxa174/bybit_bot.git
cd bybit_bot
```

### 2. Установка зависимостей
```bash
pip install pybit python-telegram-bot pandas numpy torch python-dotenv scikit-learn
```

### 3. Настройка `.env`
```bash
cp .env.example .env
```

Заполните `.env`:
```env
BYBIT_API_KEY    = "ваш_api_ключ"
BYBIT_API_SECRET = "ваш_api_секрет"
TELEGRAM_TOKEN   = "токен_telegram_бота"
ALLOWED_CHAT_IDS = "ваш_chat_id"  # опционально, для ограничения доступа
```

### 4. Запуск
```bash
python bybit_grid_bot.py
```

## 🧠 Обучение LSTM модели

### 1. Запуск тренировки
```bash
python train_lstm.py
```

Скрипт:
- Скачивает 8000 свечей (15m) с Bybit mainnet (~83 дня)
- Вычисляет 15 технических индикаторов
- Тренирует ансамбль из 3 GRU моделей
- Сохраняет чекпоинт в `lstm_ensemble.pt`

### 2. Параметры тренировки
Можно изменить в `train_lstm.py`:

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| `TOTAL_CANDLES` | `8000` | Количество свечей для обучения |
| `SEQ_LEN` | `60` | Длина последовательности |
| `N_MODELS` | `3` | Количество моделей в ансамбле |
| `EPOCHS` | `40` | Макс. эпох тренировки |
| `BATCH_SIZE` | `128` | Размер батча |
| `LEARNING_RATE` | `0.001` | Скорость обучения |
| `HIDDEN_SIZE` | `64` | Размер скрытого слоя |
| `NUM_LAYERS` | `2` | Количество GRU слоёв |
| `LABEL_HORIZON` | `12` | Горизонт предсказания (свечи) |

### 3. Фичи модели
1. `ret1`, `ret3`, `ret6` — доходности за 1/3/6 периодов
2. `rsi` — Relative Strength Index
3. `stoch_rsi` — Stochastic RSI
4. `williams_r` — Williams %R
5. `cci` — Commodity Channel Index
6. `macd_hist` — MACD Histogram
7. `bb_width`, `bb_pos` — Bollinger Bands
8. `obv_norm` — On-Balance Volume (нормализованный)
9. `momentum` — Momentum (10 периодов)
10. `atr_ratio` — ATR / Price
11. `ma_trend` — MA20 - MA50 тренд
12. `vol_ratio` — Отношение объёма к среднему

## ⚠️ Предупреждения

- **Тестовая сеть** — бот настроен на Bybit Testnet. Для реальных торгов измените `testnet=True` на `testnet=False`
- **Риск потерь** — grid trading не гарантирует прибыль. Используйте только те средства, которые готовы потерять
- **API ключи** — никогда не коммитьте `.env` с реальными ключами
- **Плечо** — высокое плечо увеличивает как прибыль, так и убытки
